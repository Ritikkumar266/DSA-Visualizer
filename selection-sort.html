Selection Sort

← Back to Sorting Algorithms

How It Works

Selection Sort works by repeatedly finding the minimum element from the unsorted part of the array and putting it at the beginning. The algorithm maintains two subarrays: the subarray which is already sorted, and the remaining subarray which is unsorted.

Performance

  • Worst-case time complexity: O(n²)
  • Average time complexity: O(n²)
  • Best-case time complexity: O(n²)
  • Space complexity: O(1)

Step-by-Step

  1. Set the first element as the minimum.
  2. Iterate through the unsorted part of the array to find the true minimum element.
  3. Swap the new minimum element with the first element of the unsorted part.
  4. The first part of the array is now sorted.
  5. Repeat the process for the remaining unsorted part of the array.

Code Implementation

def selection_sort(arr):
    n = len(arr)
    # One by one move boundary of unsorted subarray
    for i in range(n):
        # Find the minimum element in remaining
        # unsorted array
        min_idx = i
        for j in range(i+1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        # Swap the found minimum element with
        # the first element
        arr[i], arr[min_idx] = arr[min_idx], arr[i]

class SelectionSort {
    void sort(int arr[]) {
        int n = arr.length;
        // One by one move boundary of unsorted subarray
        for (int i = 0; i < n-1; i++) {
            // Find the minimum element in unsorted array
            int min_idx = i;
            for (int j = i+1; j < n; j++) {
                if (arr[j] < arr[min_idx]) {
                    min_idx = j;
                }
            }
            // Swap the found minimum element with the first
            int temp = arr[min_idx];
            arr[min_idx] = arr[i];
            arr[i] = temp;
        }
    }
}

void selectionSort(int arr[], int n) {

        // Find the minimum element in unsorted array
        int min_idx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx]) {
                min_idx = j;
            }
        }
        // Swap the found minimum element with the first
        swap(arr[min_idx], arr[i]);

}

void swap(int *xp, int *yp) {
    int temp = *xp;
    *xp = *yp;
    *yp = temp;
}
