What is a Graph?

In computer science, a graph is a non-linear data structure consisting of a set of **vertices** (or nodes) and a set of **edges** that connect these vertices. Unlike trees, graphs can have cycles, and the connections between nodes can be much more complex. They are used to model relationships between objects.
