Insertion Sort

← Back to Sorting Algorithms

How It Works

Insertion Sort is a simple, intuitive sorting algorithm that builds the final sorted array one item at a time. It iterates through the input elements and, for each element, it "inserts" it into its correct position in the already-sorted part of the array.

Performance

  • Worst-case time complexity: O(n²)
  • Average time complexity: O(n²)
  • Best-case time complexity: O(n)
  • Space complexity: O(1)

Step-by-Step

  1. Start from the second element (element at index 1).
  2. This element is the "key". Compare the key with the elements before it (the sorted part).
  3. If a sorted element is greater than the key, shift it one position to the right.
  4. Repeat this shifting until you find an element smaller than or equal to the key, or you reach the beginning of the array.
  5. Insert the key in the correct position.
  6. Repeat for all other elements in the array.

Code Implementation

def insertion_sort(arr):
    # Traverse through 1 to len(arr)
    for i in range(1, len(arr)):
        key = arr[i]
        # Move elements of arr[0..i-1], that are
        # greater than key, to one position ahead
        # of their current position
        j = i-1
        while j >= 0 and key < arr[j]:
                arr[j+1] = arr[j]
                j -= 1
        arr[j+1] = key

class InsertionSort {
    void sort(int arr[]) {
        int n = arr.length;
        for (int i = 1; i < n; ++i) {
            int key = arr[i];
            int j = i - 1;
            /* Move elements of arr[0..i-1], that are
               greater than key, to one position ahead
               of their current position */
            while (j >= 0 && arr[j] > key) {
                arr[j + 1] = arr[j];
                j = j - 1;
            }
            arr[j + 1] = key;
        }
    }
}
